Load a text file (LaTeX formulas supported)

Choose a .txt file and click Load File. The file is uploaded to the server (stored under /uploads/txt/), then loaded back. The content is split into paragraphs, and each paragraph is shown in its own block with special characters handled safely. Any LaTeX formulas (e.g. \[ ... \], $$ ... $$, \( ... \), or $ ... $) will be rendered.
